<h1 align="center">Juego de Rompecabezas en C/C++</h1>
<p>Este repositorio contiene mi código fuente y los archivos relacionados con el proyecto final desarrollado en la clase de Programación 2. El proyecto es un emocionante juego de rompecabezas que ofrece dos niveles de dificultad, funciones de inicio de sesión y registro, y utiliza archivos para gestionar información como puntajes. Desarrollado en lenguajes C y C++, el proyecto destaca la manipulación avanzada de archivos, el manejo de listas y colas, así como la implementación de sólidas validaciones para garantizar la integridad y la experiencia del usuario.</p>

<h2 align="center">Descripción del Proyecto</h2>
<p>El objetivo principal de este proyecto fue aplicar conceptos avanzados de programación, enfocándose en la manipulación eficiente de archivos para almacenar y gestionar información relevante del juego, la implementación de sistemas de inicio de sesión y registro, y el diseño de algoritmos para un juego interactivo de rompecabezas.</p>
<hr>
<h2 align="center">Características Principales</h2>
<ul>
  <li><b>Manipulación Avanzada de Archivos:</b> El juego utiliza archivos para almacenar y recuperar información vital, como puntajes y datos de usuarios, mejorando la persistencia de los datos entre sesiones de juego.</li>
  <li><b>Niveles de Dificultad:</b> Ofrece dos niveles de dificultad para adaptarse a jugadores de distintas habilidades, brindando una experiencia desafiante y entretenida.</li>
  <li><b>Inicio de Sesión y Registro:</b> Implementa un sistema de inicio de sesión y registro para personalizar la experiencia del usuario y almacenar datos de manera segura.</li>
  <li><b>Validaciones Robustas:</b> Incorpora validaciones exhaustivas para evitar repeticiones en los nombres de usuario, garantizar la integridad de los datos y proporcionar una experiencia de usuario sin errores.</li>
</ul>
<hr>
<h2 align="center">Instrucciones de Uso</h2>
<ul>
  <li><b>Compilación:</b> Utiliza un compilador de C para compilar el código fuente. Por ejemplo, puedes usar GCC con el siguiente comando: 'gcc rompecabezasmitodologiamexicana.c -o rompecabezasmitodologiamexicana'.</li>
  <li><b>Ejecución:</b> Ejecuta el programa resultante con './rompecabezasmitodologiamexicana' en la línea de comandos.</li>
  <li><b>Inicio de Sesión o Registro:</b> Sigue las instrucciones en la consola para iniciar sesión con una cuenta existente o registrar una nueva.</li>
  <li><b>Selecciona el Nivel de Dificultad:</b> Elige entre los dos niveles de dificultad disponibles antes de comenzar el juego.</li>
  <li><b>¡Diviértete Jugando!</b> Disfruta del desafío del rompecabezas y compite por los mejores puntajes.</li>
</ul>
